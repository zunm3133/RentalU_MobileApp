# RentalU

RentalU is an Android application that allows users to manage rental listings — add, view, edit, and delete rental data — with secure account creation and password recovery.

## Features

1. **Add Rental Data** – Users can create new rental listings with relevant details.
2. **View Listings** – All inserted rental data is displayed in a browsable list.
3. **Edit Listings** – Users can update existing rental data whenever changes occur.
4. **Delete Listings** – Users can remove rental data that is no longer available.
5. **Account Management** – Users can create their own account, and recover their password if forgotten.

## Tech Stack

- **Language:** Java
- **IDE:** Android Studio
- **Database:** SQLite
